# Resume Pipeline


## Summary
The job market is already tough for entry level people so I decided to setup a pipeline to help me web scrap job boards to store the results in SQL database, then use Machine learning either OpenAI API or HuggingFace API to compare the job board post with my resume, matching keywords 


## Objective 
 - [ ] Create a Python script to web scrap jobs from job boards.  (In Progress)
 - [ ] Set up a SQL Database to store the results, organizedd by title, company, pay, etc.
 - [ ] Use Machine Learning ( OpenAI API or H) to compare job board keywords against my resume. 
 - [ ] Create a  ranking system to rank each job post from 0.0 to 0.1(with 0.1 being the strongers) to determine which job I should apply to with a higher chance of getting a job. 
 - [ ] Create pipeline that will automate the process allow me to simplying some of repeated task.
